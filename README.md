# ❤️ Heart Disease Prediction Using Machine Learning

## IBM SkillsBuild CSRBOX Internship Project

### Project Overview

This project predicts whether a patient is likely to have heart disease using machine learning algorithms.

The objective is to analyze medical data, identify important health indicators, and compare different classification models to determine the best-performing algorithm.

---

## Problem Statement

Heart disease is one of the leading causes of death worldwide.

Machine Learning can assist healthcare professionals by predicting heart disease based on patient health parameters.
